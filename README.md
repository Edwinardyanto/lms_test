## ğŸ“ Customization

### Ganti Warna:
Edit CSS variables di `<style>`:
```css
:root {
    --primary: #00ff88;  /* Warna utama */
    --bg-dark: #0a0e1a;  /* Background */
    --accent-red: #ff4444; /* Accent merah */
}
```

### Ganti Font:
1. Pilih font di Google Fonts
2. Ganti link di `<head>`
3. Update `font-family` di CSS

### Ganti Konten:
- Edit text langsung di HTML
- Semua content ada di semantic tags
- Easy to find dengan search (Ctrl+F)
